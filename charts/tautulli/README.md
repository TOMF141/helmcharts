# Tautulli Helm Chart

This chart deploys [Tautulli](https://tautulli.com/).

*   **Chart Name:** tautulli
*   **Chart Version:** 0.1.6
*   **App Version:** latest
*   **Description:** A Helm chart to deploy Tautulli - a monitoring and tracking tool for Plex Media Server
