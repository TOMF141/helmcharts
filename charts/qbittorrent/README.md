# qBittorrent Helm Chart

This chart deploys [qBittorrent](https://www.qbittorrent.org/).

*   **Chart Name:** qbittorrent
*   **Chart Version:** 0.1.0
*   **App Version:** latest
*   **Description:** A Helm chart to deploy qBittorrent
